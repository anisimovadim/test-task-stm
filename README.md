# Тестовое задание на позицию Junior FrontEnd Developer

## Описание проекта

Приложение отображает таблицу пользователей с фильтром по имени и интерактивными изображениями.

### Особенности
- **Загрузка данных:** при открытии страницы данные загружаются с сервера, показывается индикатор загрузки.
- **Фильтрация:** поиск пользователей по имени прямо на клиенте; если ничего не найдено — показывается сообщение.
- **Оптимизация ввода:** поле фильтра обновляется с `debounce`, чтобы избежать лишних перерисовок (без `useState`).
- **Тултипы:** при наведении на картинку появляется изображение высокого разрешения.

### Технологии
- HTML / CSS / TypeScript
- React
- Client-side rendering

## Инструкция к запуску

1. Клонируйте репозиторий:
```bash
git clone https://github.com/anisimovadim/test-task-stm.git
```

2. Перейдите в папку проекта:
```bash
cd test-task-stm
```
3. Установите зависимости:
```bash
npm install
```
4. Запустите проект:
```bash
npm run dev
```
